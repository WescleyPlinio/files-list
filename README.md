# Files list
O files list é um projeto pessoal que consiste numa página de upload e listagem de arquivos.

## Tecnologias
- HTML e CSS
- Tailwind CSS
- React e TypeScript

## Preparação do ambiente
### Back-end
````
cd ./backend
npm install
npm run start:dev
````

### Front-end
````
cd ./frontend/files-site
npm install
npm run dev
````

## Banco de dados
### É necessário criar uma conta no [Neon](https://neon.com/) e seguir os passos:
1. Quando autenticado, acesse o [dashboard](https://console.neon.tech) e clicar em "New project"
2. Ao criar o servidor do banco de dados, clique em "Connect"
3. Troque a opção "Connection string" por "NestJS" e copie o conteúdo.
4. Crie o arquivo .env na pasta /backend e cole o conteúdo
