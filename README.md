# Files list
O files list é um projeto pessoal para reforçar e aprender novos conhecimentos.

## Tecnologias
- HTML e CSS
- Tailwind CSS
- React e TypeScript

## Iniciando o projeto
### Back-end
````
cd ./backend
npm install
npm run start:dev
````

### Front-end
````
cd ./frontend/files-site
npm install
npm run dev
````
